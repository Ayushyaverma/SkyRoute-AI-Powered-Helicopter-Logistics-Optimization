# Screenshot Capture Guide

Save each screenshot as a `.png` in this folder with the exact filename listed below.
Use a **1280x800** or wider browser window for consistency.

## How to Get Good Screenshots

1. **Populate data first**: Create 5–10 demands across different clients, approve them as admin, run the optimizer, then capture
2. **Use the full workflow**: Login → Create demands → Approve → Optimize → View schedules → Enter actuals
3. **Crop browser chrome**: Show just the app content, not the browser URL bar / tabs
4. **Dark mode**: The app is dark-themed — make sure screenshots aren't washed out
